## 1. Розбиття `FormsPanel.tsx`

Зараз цей файл виконує роль "головного контролера" для всього лівого сайдбару. В ньому знаходяться всі форми, парсери імпорту, логіка побудови геометрії та редактори профілів.

### 🗂 Нова структура файлів (`src/components/forms/`)
Ми створимо нову директорію `src/components/forms` і винесемо туди логіку з `FormsPanel.tsx`.

#### A. Форми деталей (Shapes)
Кожна геометрична форма має свої специфічні поля (наприклад, глибина для Г-подібної, радіус для круглої).
- `[NEW]` **`RectShapeForm.tsx`** — Форма для прямокутних деталей.
- `[NEW]` **`LShapeForm.tsx`** — Форма для Г-подібних деталей.
- `[NEW]` **`UShapeForm.tsx`** — Форма для П-подібних деталей.
- `[NEW]` **`RoundShapeForm.tsx`** — Форма для круглих/овальних деталей.
- `[NEW]` **`SinkShapeForm.tsx`** — Форма для мийок (щілинна/прямокутна).

#### B. Імпорт та Інтеграції
- `[NEW]` **`DxfImportPanel.tsx`** — Логіка імпорту DXF файлів (парсинг ліній, вибір шарів).
- `[NEW]` **`ApprovalImportPanel.tsx`** — Логіка імпорту "Узгоджень" (форматів замовника).

#### C. Редактори кромок та налаштування
- `[NEW]` **`EdgeProfileEditor.tsx`** — Вибір профілю кромки (A, B, C, D) та підворотів.
- `[NEW]` **`GeneralSettingsPanel.tsx`** — Загальні налаштування допусків та відступів.

#### D. Головний компонент
- `[MODIFY]` **`FormsPanel.tsx`** — Буде сильно спрощений. Стане просто роутером/контейнером, який рендерить потрібний під-компонент залежно від `activeTab` або `selectedDetail.shape`.

## 2. Розбиття `SlabBoard.tsx`

`SlabBoard.tsx` — це інтерактивний SVG-канвас. Наразі він змішує в собі рендер слябів, логіку Drag & Drop, логіку відображення розмірів (dimension hints) та лупу (magnifier).

### 🗂 Нова структура файлів (`src/components/2d/`)

#### A. Хуки для складної логіки
Щоб відокремити UI від бізнес-логіки перетягування та виділення, створимо кастомні хуки.
- `[NEW]` **`hooks/usePlacementDrag.ts`** — Обробка `onPointerDown`, `onPointerMove`, `onPointerUp` для перетягування деталей по слябу.
- `[NEW]` **`hooks/useSelectionBox.ts`** — Логіка прямокутного виділення декількох деталей (shift + drag).

#### B. Візуальні компоненти
- `[NEW]` **`SlabCanvas.tsx`** — Чистий SVG-контейнер, що відповідає лише за масштабування та panning.
- `[NEW]` **`PlacementPart.tsx`** — Компонент, який малює одну розміщену деталь на слябі (зараз це функція всередині SlabBoard).
- `[NEW]` **`SlabMagnifier.tsx`** — Вікно-лупа, що з'являється при наведенні або перетягуванні на краях деталі.
- `[NEW]` **`SlabDimensionHints.tsx`** — Відображення розмірних ліній та відступів між деталями на канвасі.
- `[NEW]` **`ManualDimensionLine.tsx`** — Відмальовка користувацьких лінійок, які можна малювати поверх слябу.

#### C. Головний компонент
- `[MODIFY]` **`SlabBoard.tsx`** — Стане просто збіркою (assembly) вищеописаних компонентів та передаватиме їм стан через хуки.
