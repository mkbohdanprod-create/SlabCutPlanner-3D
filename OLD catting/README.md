# SlabCutPlanner

Локальний браузерний застосунок для попереднього розкрою слебів листового матеріалу.

## Запуск

```bash
npm install
npm run dev
```

## Що вже реалізовано

- React + TypeScript + Vite
- модульна структура
- централізований state через Zustand
- локальне автозбереження в localStorage
- JSON import / export
- демо-додавання слебів і деталей
- базова геометрія для прямокутних, Г-подібних, П-подібних, круглих, овальних деталей
- евристичний packing engine з автоповоротом 0/90/180/270
- ручне перетягування деталей
- manual lock
- детекція конфліктів і виходу за межі
- SVG-візуалізація
- експорт PNG поточного SVG
- базовий фінальний PDF
