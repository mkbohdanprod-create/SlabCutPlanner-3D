# Інструкція користувача SlabCutPlanner

SlabCutPlanner — локальний браузерний застосунок для попереднього розкрою слебів, розміщення деталей, підбору текстури та підготовки експорту.

## 1. Верхня панель

У верхній панелі заповнюються основні дані замовлення:

- **Номер замовлення** — внутрішній номер або код проєкту.
- **Контрагент** — клієнт або замовник.
- **Мова** — перемикання інтерфейсу між українською, англійською та польською.
- **Підбір текстури** — вмикає зону підбору текстури для деталей.
- **Перерахувати** — запускає автоматичний розкрій.
- **Стрілка біля Перерахувати** — вибір режиму розкрою.
- **Очистити розрахунок** — очищає слеби, фото, деталі та поточний розрахунок для нового проєкту.

Під верхньою панеллю показується короткий статус: результат розкрою, площа заготовок, площа деталей, кількість слебів і дата оновлення.

## 2. Слеби

Слеб додається в блоці **Додати slab**.

Можна вказати:

- серійний номер або назву;
- матеріал;
- ширину, висоту, товщину;
- мінімальний відступ;
- декор і коментар.

Є швидкі кнопки типових розмірів. Після додавання слеб з’являється у робочій зоні.

Для редагування слеба натисніть правою кнопкою миші по порожньому місцю слеба і виберіть **Редагувати слеб**. Зміна параметрів не видаляє деталі, фото, дефекти або ручне розміщення.

## 3. Фото, дефекти та інспектор слеба

У блоці **Інспектор слеба** можна вибрати активний слеб і налаштувати фото:

- масштаб фото;
- зміщення X/Y;
- прозорість;
- поворот фото на 90°.

Дефекти додаються через кнопку **Дефект** або через контекстне меню слеба. Для кожного дефекту можна вибрати тип і розмістити його на слебі. Деталі, які заходять у дефект, підсвічуються як конфліктні.

## 4. Деталі

У блоці **Деталі** доступні дії:

- **Додати деталь** — відкриває конструктор деталей.
- **Імпортувати DXF** — відкриває попередній перегляд контурів з DXF.
- **Припуски** — відкриває технічні параметри припусків.

У конструкторі можна вибрати тип і форму деталі, ввести розміри, кількість, потовщення та підвороти. Для потовщень і підворотів можна вибрати окремі сторони або увімкнути всі сторони одразу.

У списку деталей доступні дії **Редагувати** і **Видалити**.

## 5. Режими розкрою

Режим вибирається біля кнопки **Перерахувати**.

- **Економний** — компактне розміщення з пріоритетом економії матеріалу.
- **Оптимальний** — намагається тримати логічно пов’язані частини на одному слебі.
- **Повна текстура** — розміщує деталь і її елементи за правилами переходу текстури: потовщення біля своєї сторони, підворот біля відповідного потовщення.

Автоматичний розкрій запускається тільки кнопкою **Перерахувати**.

## 6. Ручна робота зі слебом

Деталі можна переміщувати мишкою:

- перетягувати по слебу;
- переносити між слебами;
- переносити в буфер нерозміщених деталей;
- повертати подвійним кліком або через контекстне меню **Кут**.

Групове виділення:

- затисніть ліву кнопку миші й обведіть зону рамкою;
- або затисніть **Ctrl** і клікайте по потрібних деталях.

Виділену групу можна переміщувати разом. Клавіші зі стрілками рухають виділення на малий крок, **Ctrl + стрілка** — на більший крок. **Delete** прибирає вибрані деталі або елементи у буфер нерозміщених.

## 7. Контекстне меню деталі

Правий клік по деталі відкриває меню:

- **Кут** — задати точний кут повороту.
- **Прив’язати / Відв’язати** — закріпити деталь за конкретним слебом.
- **Зафіксувати / Зняти фіксацію** — заборонити автоматично рухати конкретну деталь.
- **Зафіксувати збірку / Зняти фіксацію збірки** — зафіксувати основну деталь разом з її потовщеннями, підворотами та іншими елементами.
- **Редагувати** — відкрити параметри деталі.
- **Копіювати** — створити копію.
- **Видалити** — видалити деталь.

Зафіксована збірка залишається на тому самому слебі, з тими самими координатами та поворотом під час перерахунку.

## 8. Додаткові розміри

Додатковий розмір потрібен для ручного контролю відстані між деталями або від деталі до краю слеба.

Як створити:

1. Правою кнопкою миші натисніть по слебу.
2. Виберіть **Додатковий розмір**.
3. Поставте перший клік — початкова точка.
4. Поставте другий клік — кінцева точка.

Біля кнопки **Розміри** є дії:

- **Очистити додаткові розміри** — видаляє всі додаткові розміри активного слеба.
- **Видалити розмір** — видаляє вибраний додатковий розмір.

Один розмір можна вибрати кліком по його лінії або підпису, після чого видалити кнопкою **Delete**.

## 9. Зона підбору текстури

Зона з’являється, коли у верхній панелі увімкнено **Підбір текстури**.

У ній можна:

- бачити деталі з фрагментами текстури зі слебів;
- переміщувати деталі для підбору композиції;
- вмикати **Показати елементи**;
- масштабувати зону через **Shift + колесо миші**;
- відкривати окреме прев’ю;
- створювати рамки для експорту підбору текстури в PDF.

Правий клік по деталі в зоні підбору текстури відкриває меню для повороту або задання точного кута.

## 10. Імпорт та експорт

У блоці **Швидкі дії** доступно:

- **Зберегти проект** — завантажує файл проєкту у JSON.
- **Завантажити проект** — відкриває раніше збережений JSON.
- **Експортувати PNG** — формує зображення слебів.
- **Експортувати PDF** — відкриває вікно налаштувань PDF.

У PDF можна вибрати формат сторінки, масштаб, режими відображення, списки деталей, дефекти, коментарі та зону підбору текстури.

## 11. DXF-імпорт

Кнопка **Імпортувати DXF** відкриває файл DXF і показує попередній перегляд контурів.

У вікні попереднього перегляду можна:

- змінити назву контуру;
- вибрати тип і форму;
- призначити роль: деталь, потовщення або підворот;
- для елементів вибрати прив’язку до конкретної деталі та сторони;
- видалити зайвий контур перед імпортом.

Після підтвердження імпорту контури додаються в проєкт як звичайні деталі або елементи й беруть участь у розкрої.
